入门指南

Starter Kit、模型环境和安全提交格式

模型与环境

统一基础模型为 Hugging Face openai/clip-vit-base-patch32，视觉输入为 224×224，patch size 为 32，投影特征维度为 512。

from transformers import CLIPModel model = CLIPModel.from_pretrained(MODEL_PATH) model.eval()

Starter Kit

Starter Kit 将提供模型加载、0/1 检测结果生成、embedding 归一化、JSON 模板、safetensors 写入、提交校验、ZIP 打包和参考基线。

仓库地址、依赖版本、数据大小、校验值、推荐磁盘空间、参考硬件和预计运行时间将在数据发布前公布。

safetensors 示例

import torch from safetensors.torch import save_file embedding = embedding.detach().to("cpu", dtype=torch.float32).reshape(512) embedding = embedding / embedding.norm(p=2).clamp_min(1e-12) save_file({"model_0001": embedding}, "embeddings.safetensors")

实际提交可在同一个 embeddings.safetensors 中保存多个以 model_id 命名的向量。

参赛流程

  1. 注册并创建队伍。
    每支队伍最多 5 人。
  2. 下载模型和 Starter Kit。
    每阶段包含 200 个模型。
  3. 生成检测结果。
    为全部模型提交整数标签 0 或 1。
  4. 生成目标 embedding。
    只为预测为后门的模型生成 512 维标准化向量。
  5. 校验并打包。
    ZIP 仅包含 predictions.json 和 embeddings.safetensors。
  6. 上传提交。
    排行榜仅显示最终总分。
